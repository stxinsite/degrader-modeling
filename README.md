# degrader-modeling
Mathematical modeling of protein degrader binding equilibria. 

Mathematical details can be found [here](https://www.overleaf.com/read/rkzvrfxwdjbg)

## Dependencies
### Python dependencies
- numpy
- pandas
- scipy
- lmfit
- pymc3
- theano
- arviz
- seaborn

### R dependencies
- nleqslv
- cmdstan
- posterior
- bayesplot
- shinystan
- dplyr
- ggplot2
